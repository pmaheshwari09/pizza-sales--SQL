Pizza Sales Analysis using SQL
This project involves analyzing a pizza sales dataset using SQL to extract meaningful business insights. The dataset includes information such as order details, pizza types, sizes, quantities, and prices.
Objectives:

Analyze total revenue and order trends over time
Identify best-selling and least-selling pizzas by category and size
Calculate average order value and peak ordering hours
Determine the most profitable pizza categories
Rank pizzas based on revenue contribution using window functions

Key SQL Concepts Used:

Joins (to combine order, pizza, and pricing tables)
Aggregate functions (SUM, COUNT, AVG)
Window functions (RANK, ROW_NUMBER, running totals)
Common Table Expressions (CTEs) for cleaner, modular queries
Group By and Having clauses for filtered aggregations

Outcome:
The analysis helped identify sales patterns, top-performing products, and revenue drivers, providing actionable insights for business decision-making.
